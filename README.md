# Smille - E-commerce de Moda para Festivais & Raves 🌅✨

A **Smille** é uma loja online fictícia de alta fidelidade especializada em vestuário e acessórios para festivais de música eletrônica e raves. O projeto foi projetado com uma identidade visual humana, orgânica e sofisticada, inspirada no clima de pôr do sol e desertos (estilo *boho-chic* e utilitário), utilizando cores quentes terrosas e tipografia elegante.

---

## 🎨 Identidade Visual e Design

*   **Paleta de Cores**: Tons quentes de pôr do sol, como dourado (`#e2a87c`), terracota (`#cc7f61`), argila (`#d9695b`) e fundo carvão escuro aquecido (`#12100e`).
*   **Tipografia**: Combinação clássica e editorial com a fonte serifada *Playfair Display* para títulos e títulos conceituais, e a moderna sans-serif *Plus Jakarta Sans* para legibilidade do corpo.
*   **Efeito Vidro (Glassmorphism)**: Cartões de produto e painéis com fundo semi-transparente fosco e desfoque suave para simular sofisticação.

---

## 🚀 Funcionalidades Principais

1.  **Marquee de Anúncios (Promo Bar)**: Faixa rolável infinita no topo do site para exibir anúncios e cupons ativos.
2.  **Carrossel de Slideshow (Hero)**: Banner animado automático com transições suaves de fade entre imagens editoriais de festivais.
3.  **Grade de Diferenciais (Features)**: Destaque rápido das vantagens de compra como Frete Grátis, Parcelamento e Trocas.
4.  **Grade de Coleções**: Navegação intuitiva e rápida dividida entre os departamentos *Feminino*, *Masculino* e *Acessórios*.
5.  **Catálogo Dinâmico (Filtros)**:
    *   Filtragem por categoria de produto (roupas, conjuntos, acessórios).
    *   Filtro de preço máximo em tempo real.
    *   Ordenação por preço, destaques e popularidade.
    *   Busca inteligente de produtos.
6.  **Modal com Miniaturas Interativas**: Ao espiar um look, é possível escolher tamanhos (PP ao GG) e alternar entre as fotos de frente e verso da roupa nas miniaturas.
7.  **Carrinho Lateral (Cart Drawer)**: Painel dinâmico para gerenciar quantidades, excluir itens e atualizar o subtotal da compra.
8.  **Checkout Simulado Passo a Passo**: Processo de compra fictício em 3 etapas (Dados de Envio, Forma de Pagamento com Pix/Cartão/Boleto e tela de confirmação do pedido).

## 💻 Como Rodar o Projeto Localmente

Siga os passos abaixo para iniciar a aplicação na sua máquina:

### 1. Clonar o Repositório
```bash
git clone https://github.com/Dev-Jhonsom/smille-ecommerce.git
cd smille-ecommerce
```

### 2. Instalar Dependências
```bash
npm install
```

### 3. Iniciar Servidor de Desenvolvimento
```bash
npm run dev
```
Abra o navegador em `http://localhost:5173/` para ver e interagir com o site.

### 4. Build de Produção
Para compilar e otimizar os arquivos estáticos para publicação:
```bash
npm run build
```
Os arquivos prontos para hospedagem gratuita (Vercel, Netlify, etc.) serão salvos na pasta `/dist`.
